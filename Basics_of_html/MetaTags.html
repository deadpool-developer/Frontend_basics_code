META TAGS IN HTML

meta name="description" content="Day5"

name="description" is used for the description of the html page that is shown when the user searches for the website.

meta name="keywords" content="HTML, CSS"

This name = "keyword" is used so that when a user searches for a content that is available on our website, google shows our website according to that particular keyword.

meta name="author" content="Aditya Saini"

This is the author of the website

meta name="viewport" content="width=device-width, initial-scale=1.0"

name="viewport" is used to give the responsiveness to our website when it is used on different devices content="width=device-width is used to tell the browser toadjust the web[age size according to the wodht of the device initial-scale=1.0" this is used for the zoom purpose of our webpage

meta name="language" content="en"

The google generally stores the website in english language and if any website is written in other language then the google shows for the language change option of the website itself.

meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1

This meta tag states that on which browser our website will run properly.

meta http-equiv="Cache-Control" content="no-cache"

generally Google cache the data that needs to be GET everytime when its www.google.com is accessed eg, our photo This is done to avoid the Extra APIs call to get the data everytime and this is why our photo is shown even if the internet bandwidth is low Websites like Facebook & Instagram do not do this cache of data because if they do then everytime older posts will be shown to the user and no new post will be shown content="no-cache" this is passed in the attribute so that our website do not cache any data and always new data is shown to the user

meta charset="UTF-8"

Previously in HTML4 there were 256 character but now HTML5 have only 128 ASCII characters To use emojis or any other entities like alpha,beta,gamma etc we use UTF-8 so that our browser can easily read that particular emoji and show that on our webpage
